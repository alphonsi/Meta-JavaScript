# Meta JavaScript 📜

This repository contains my coursework, exercises, projects, and assignments for the **JavaScript** module of the **Meta Full-Stack Developer Professional Certificate** on Coursera.

## 📚 Course Overview

This module covers JavaScript programming fundamentals and advanced concepts essential for both front-end and back-end development, including ES6+ features, asynchronous programming, DOM manipulation, and more.

## 📁 Repository Structure

```
Meta-JavaScript/
├── exercises/         # Practice exercises and coding challenges
├── projects/          # JavaScript-based mini-projects
├── labs/              # Guided lab work
└── notes/             # Personal notes and summaries
```

## 🧠 Key Topics Covered

- JavaScript Basics (Variables, Data Types, Operators)
- Control Flow (Conditionals, Loops)
- Functions & Scope
- Arrays & Objects
- ES6+ Features (Arrow functions, Destructuring, Spread/Rest, Modules)
- DOM Manipulation & Events
- Asynchronous JavaScript (Callbacks, Promises, Async/Await)
- Error Handling & Debugging
- Object-Oriented JavaScript
- Functional Programming Concepts
- Working with APIs (Fetch, AJAX)
- JavaScript Testing

## 🛠️ Technologies Used

- JavaScript (ES6+)
- Node.js
- HTML5 & CSS3
- Browser DevTools

## 🎯 Goals

- Master JavaScript fundamentals and modern syntax
- Write clean, efficient, and maintainable code
- Understand asynchronous programming patterns
- Build interactive, dynamic web applications
- Prepare for the Meta JavaScript certification assessment
